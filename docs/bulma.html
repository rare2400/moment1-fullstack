Bulma

Länk till Bulmas webbplats

Bulma är ett ramverk som är komponentbaserat med färdiga UI-komponenter för att styla webbplatser. Komponenterna hämtas med hjälp av klasser i HTML-elementen, vilket ger färdiga utseenden till exempelvis knappar, tabeller och formulär. Utvecklingen med Bulma var relativt enkel, främst eftersom jag har tidigare erfarenhet av att använda komponentbaserade ramverk. Dokumentationen var lätt att följa och tydligt strukturerad för att hitta de komponenter man ville använda. Det kräver ingen direkt CSS-kunskap och klasserna som används är beskrivande för vilken komponent de är till för. Fördelen med att ha kunskaper om CSS däremot är att man vid behov kan göra anpassningar. Att utveckla med ramverket Bulma gick relativt snabbt och själva innehållet var enkelt att skapa.

En del som jag fastnade på var stylingen av headern och navigeringen, då jag är rätt inkörd på att ha en header med navigeringen inuti. Med Bulma så är headern "navbar", vilket var förvirrande för mig innan jag förstod det. Själva navigeringen däremot med hamburgermeny för mobil och utskriven navigering för större skärmer var mycket enkel att implementera. Det var skönt och underlättade mycket då det annars är något som tar lite längre tid att utveckla.

Eftersom Bulma är ett komponentbaserat CSS-ramverk så har det en tydlig stil som standard där sidor kan bli väldigt lika varandra grafiskt sett. Det finns möjlighet att ändra färger och anpassa viss styling med variabler men det har inte samma frihet som utilitybaserade ramverk. Utvecklingen med denna uppgift har jag endast använt befintliga komponenterna och sedan anpassat med padding och marginaler. I framtiden hade jag velat testa att ta hjälpa av SASS tillsammans med ramverket för att kunna anpassa stylingen mer.

Bulma är ett ramverk baserat på ren CSS utan JavaScript, vilket gör det bra presterande och lassar endast inte en enda CSS-fil. När denna sida kontrolleras med Googles DevTools Lighthouse får den 100 poäng för performance. Användarna av Bulma samlas på olika plattformar som Github och DEV där man kan se andra utvecklares lösningar, instruktioner och ställa frågor. Den senaste versionen som släpptes är v.1.0.4 och uppdateras inte så ofta. Dess pålitlighet som ramverk kan därmed ifrågasättas när man ser på det långsiktigt och vid utveckling av större projekt. Trots att detta har har Bulma relativt många användare och hyllas ofta av dess användare för sin enkelhet vid användning.
